# e-Ticket — Compra de Ingressos

Projeto desenvolvido em **`HTML, CSS, JavaScript`** que simula a compra de ingressos para um evento, permitindo selecionar o tipo de ingresso, informar a quantidade desejada e atualizar dinamicamente a disponibilidade de cada setor.  

---

## Funcionalidades

- Seleção do tipo de ingresso
- Informar a quantidade de ingressos desejada
- Verificação de disponibilidade antes da compra
- Atualização dinâmica da quantidade disponível
- Exibição de mensagens de sucesso ou erro
- Controle de estoque diretamente na interface
